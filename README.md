## About the Project

A pet project for the Taberna Saloia restaurant, based in Loures, Portugal. Built to provide a fast, appealing, and responsive experience for users, utilizing a modern ecosystem based on **React** and **Vite**. A large part of the code was AI-generated, using the **Google Gemini** models to improve a **Lovable.ai** generated template.

## Tech Stack

- **[React](https://react.dev/)** + **[TypeScript](https://www.typescriptlang.org/)** - Application core
- **[Vite](https://vitejs.dev/)** - Build tool
- **[Tailwind CSS](https://tailwindcss.com/)** - Styling
- **[Shadcn UI](https://ui.shadcn.com/)** & **[Radix UI](https://www.radix-ui.com/)** - UI Components
- **[React Router](https://reactrouter.com/)** - Routing
- **[React Query](https://tanstack.com/query/latest)** - State management

## How to Run Locally

1. **Clone the repository**

   ```bash
   git clone <repository-url>
   cd saloia-showcase
   ```

2. **Install dependencies**
   Using `npm`, `yarn`, or `bun` is recommended:

   ```bash
   npm install
   ```

3. **Start the development server**
   ```bash
   npm run dev
   ```
